## Requirements

- WSL2 or Linux
- `bash`
- `coreutils`

## Installing

Run [`install.sh`](./install.sh) to generate an entrypoint file and **copy** them to alacritty's configuration directory.

```sh
./install.sh
```

### File placement

- Windows: `%APPDATA%\alacritty\`
- Linux: `$HOME/.config/alacritty/`
